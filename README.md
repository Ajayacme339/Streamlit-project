# GlobalJava Roasters - Customer Feedback App

A Streamlit-based customer feedback application for GlobalJava Roasters coffee shop.

## Features

- ✅ Radio button selection for coffee enjoyment
- ✅ Interactive rating slider (1-10 scale)
- ✅ Service type selection dropdown
- ✅ Customer feedback form
- ✅ Terms and conditions checkboxes
- ✅ Coffee image display

## Installation

1. Clone this repository:
```bash
git clone <your-repository-url>
cd <repository-name>
```

2. Install dependencies:
```bash
pip install -r requirements.txt
```

## Usage

Run the Streamlit app locally:
```bash
streamlit run javacoffee.py
```

The app will open in your default browser at `http://localhost:8501`

## Deployment

### Streamlit Cloud

1. Push your code to GitHub
2. Visit [Streamlit Cloud](https://share.streamlit.io/)
3. Connect your GitHub repository
4. Deploy with one click!

## CI/CD

This project includes a GitHub Actions workflow that:
- Runs PEP 8 compliance checks
- Tests Python syntax
- Validates the Streamlit app

## Requirements

- Python 3.11+
- Streamlit 1.28.0+
